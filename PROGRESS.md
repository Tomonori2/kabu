## ⚠️ メモ・配当機能を使うための作業（1回だけ）

SQL Editor で入力欄を空にしてから、以下を貼り付けて Run
（「Potential issue detected」が出たら **Run without RLS** を選ぶ）:

```sql
alter table trades add column memo text;

create table if not exists dividends (
    id     bigint generated by default as identity primary key,
    date   date not null,
    name   text not null,
    amount integer not null
);
```

## ⚠️ 目標設定・ウォッチリストを使うための作業（1回だけ・実施済み）

SQL Editor（https://supabase.com/dashboard/project/nnsctxvscnkbctylxrxs/sql/new）で
入力欄を空にしてから、以下を貼り付けて Run:

```sql
create table if not exists settings (
    key   text primary key,
    value text not null
);

create table if not exists watchlist (
    id   bigint generated by default as identity primary key,
    code text not null,
    name text not null

```

これで「🎯 月の目標」（ホーム）と「👀 ウォッチリスト」（保有タブ）が使えるようになる。

## ⚠️ 現在株価機能を使うための作業（1回だけ）

`trades` テーブルに `code` 列を追加する必要がある:

1. Table Editor を開く: https://supabase.com/dashboard/project/nnsctxvscnkbctylxrxs/editor
2. 左のリストの `trades` の横の「…」→「**Edit table**」
3. Columns の一番下の「**Add column**」→ Name: `code` ／ Type: `text`
4. 「**Save**」

これをやると、取引追加のときに証券コード（例: トヨタ=7203）を入力でき、
保有状況タブに現在株価と含み損益が表示される。
