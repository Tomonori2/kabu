## 📋 パソコンでやること（これで完成）

1. SQL Editor を開く: https://supabase.com/dashboard/project/nnsctxvscnkbctylxrxs/sql/new
   - ⚠️ **Chromeの翻訳機能はオフにする**（オンだとSupabaseの画面が壊れる）
2. 入力欄を **Ctrl+A → Delete で空にしてから**、下のSQLを貼り付けて **Run**
3. 「Potential issue detected」という警告が出たら **「Run this query」を押して実行**
   （drop table が入っているための確認。データはまだ無いので消えて困るものは無い）
4. 「Success. No rows returned」と出たら成功
5. アプリ（streamlit.app のURL）を再読み込みして、取引を1件記録 → 履歴に出たら完成 🎉

```sql
drop table if exists trades;

create table trades (
    id     bigint generated by default as identity primary key,
    date   date    not null,
    name   text    not null,
    baibai text    not null check (baibai in ('買', '売')),
    shares integer not null check (shares > 0),
    price  integer not null check (price > 0)
);

alter table trades enable row level security;

create policy "allow anon select" on trades
    for select to anon using (true);

create policy "allow anon insert" on trades
    for insert to anon with check (true);
```

（同じ内容は `schema.sql` にもある）
